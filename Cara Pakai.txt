Cara Menggunakan di localhost:

1. Install Software Xampp 1.6.2 (PHP 5)
2. Copy folder sewa yang di download ke xampp/htdocs
3. buat database di phpmyadmin (xampp) dengan nama db_rt (penyortiran utf8_bin)
4. import database db_rt
5. buka browser, ketik localhost/sesuai nama folder
6. Gunakan Login
